# Athena Server
Go-based self-host server for the Athena app.
Deployment of a server will provide the following functionality to Athena:
- User Authentication: Register your libraries to a "user" login, specific to the server.
- Multi-library syncing: Access your libraries via your user across unlimited amounts of devices!
- Library sharing: Share your libraries with trusted people!
- Real-time chat: Communicate with people that have access to your library in a library-level chat!
